**Course Code:** 214448  
**Second Year – Information Technology (2019 Course)**  
Savitribai Phule Pune University, Pune  

---

## 📌 Prerequisite
- Basic knowledge of Programming Language  

---

## 🎯 Course Objectives

1. Apply object-oriented programming concepts.  
2. Design real-life problem solutions using OOP.  
3. Develop object-oriented programming skills.  

---

## 🎓 Course Outcomes

- **CO1:** Differentiate programming paradigms.  
- **CO2:** Identify and implement classes, objects, and constructors.  
- **CO3:** Apply inheritance and polymorphism.  
- **CO4:** Handle exceptions and generic programming.  
- **CO5:** Implement file handling in applications.  
- **CO6:** Apply design patterns for OOP solutions.  

---

# 📑 List of Laboratory Assignments

---

### 1️⃣ Classes and Objects  
- Design `Complex` class  
- Default & Parameterized Constructors  
- Perform arithmetic operations  

---

### 2️⃣ Polymorphism  
- Publication, Book, Magazine classes  
- Common attributes & methods  
- Calculate total sale of publication  

---

### 3️⃣ Inheritance  
- Base class: Employee  
- Derived classes: Programmer, Team Lead, Assistant Project Manager, Project Manager  
- Generate employee payslip  

---

### 4️⃣ Dynamic Binding  
- Base class: Shape  
- Derived classes: Triangle, Rectangle  
- Compute area using abstract method  

---

### 5️⃣ Interface  
- Vehicles interface  
- Implement for Bicycle, Car, Bike  
- Common operations: Gear Change, Speed Up, Apply Brake  

---

### 6️⃣ Exception Handling  
- Handle Arithmetic Exception  
- Array Index Out of Bounds  
- Number Format Exception  

---

### 7️⃣ Template / Generics  
- Generic program using collection class  
- Count even, odd, prime, palindrome elements  

---

### 8️⃣ File Handling  
- Student Database using Files  
  - Create  
  - Display  
  - Delete  
  - Update  
  - Search  

---

### 9️⃣ Case Study (Any One)

**Banking System**  
- Create Account  
- Deposit / Withdraw  
- Balance Check  
- Account Info  

**OR**

**Inventory Management System**  
- Product List  
- Purchase / Shipping  
- Stock & Profit Calculation  

---

### 🔟 Factory Design Pattern  
- Car class  
- Subclasses: Hatchback, Sedan, SUV  
- Implement CarFactory  

---

### 1️⃣1️⃣ Strategy Design Pattern  
- Shopping Cart  
- Payment Strategies: Credit Card, PayPal, Bitcoin
